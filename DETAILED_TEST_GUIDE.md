# MetaCoding 详细测试指南

## 🌐 第一步：访问网站
1. 打开浏览器
2. 访问: https://Excellence8.github.io/metacoding/
3. 等待页面加载完成

## 🎯 第二步：测试核心功能

### 测试1：导航菜单
- [ ] 点击"🏠 首页" - 应显示欢迎页面
- [ ] 点击"🚀 生成器" - 应显示项目生成页面
- [ ] 点击"📚 模板库" - 应显示模板选择页面
- [ ] 点击"📋 项目管理" - 应显示历史记录
- [ ] 点击"⚙️ 设置" - 应显示设置页面

### 测试2：项目生成流程
1. 进入"📚 模板库"
2. 选择"React + TypeScript"模板
3. 点击"✅ 选择此模板"
4. 自动跳转到生成器页面
5. 输入项目名称（如：my-project）
6. 点击"🎯 一键生成项目"
7. 等待2秒生成完成
8. 验证生成结果显示

### 测试3：导出功能
生成项目后测试：
- [ ] 点击"📦 ZIP 项目" - 应下载ZIP文件
- [ ] 点击"📥 JSON 配置" - 应下载JSON文件
- [ ] 点击"📋 复制配置" - 应提示复制成功
- [ ] 点击"🚀 复制命令" - 应提示复制成功

### 测试4：项目管理
1. 生成几个测试项目
2. 进入"📋 项目管理"
3. 验证：
   - [ ] 历史记录显示正确
   - [ ] 项目统计更新
   - [ ] 点击"🔄 重新生成"可以重新生成
   - [ ] 点击"🗑️ 删除"可以删除项目
   - [ ] 点击"🧹 清空历史"可以清空所有

### 测试5：设置功能
1. 进入"⚙️ 设置"
2. 测试：
   - [ ] 切换"浅色主题"、"深色主题"、"蓝色主题"、"紫色主题"
   - [ ] 调整字体大小滑块
   - [ ] 切换语言（中文、英文、日文）
   - [ ] 保存设置

### 测试6：响应式测试
1. 按F12打开开发者工具
2. 点击设备工具栏（📱图标）
3. 测试不同设备：
   - [ ] iPhone SE (375x667)
   - [ ] iPad (768x1024)
   - [ ] 桌面 (1200x800)
4. 验证布局是否正常

## 🔧 第三步：技术验证

### 控制台检查
按F12  Console，检查：
- [ ] 无红色错误信息
- [ ] 只有正常日志
- [ ] 项目生成时的日志输出

### 网络检查
按F12  Network，检查：
- [ ] 资源加载正常
- [ ] 没有404错误
- [ ] 加载时间合理

### 性能检查
1. 按F12  Lighthouse
2. 运行性能测试
3. 验证：
   - [ ] 性能得分 > 80
   - [ ] 无障碍得分 > 90
   - [ ] 最佳实践得分 > 90

## 📱 第四步：多浏览器测试
在不同浏览器测试：
- [ ] Google Chrome
- [ ] Mozilla Firefox
- [ ] Microsoft Edge
- [ ] Safari (如果有)

## 🐛 第五步：错误处理测试
故意测试：
1. 不输入项目名称直接生成
2. 重复生成同一项目
3. 清空历史后查看统计
4. 断网情况下操作
