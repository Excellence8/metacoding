## 详细功能说明

### 🎯 项目生成器
- **项目名称**：只能使用字母、数字、连字符

- **一键生成**：2秒内生成完整项目结构

1. **ZIP项目包**：完整的可运行项目
2. **JSON配置**：轻量的配置文件
3. **复制功能**：快速分享和保存

### 📋 项目管理
- **历史记录**：按时间排序的所有项目
- **项目统计**：生成数量、模板分布
- **重新生成**：点击可以重新生成项目
- **删除管理**：单个删除或清空全部

### ⚙️ 设置系统
- **主题切换**：4种配色方案
- **语言设置**：中/英/日文
- **字体大小**：调整界面文字大小
- **自动保存**：设置自动保存到本地

- `Ctrl + T`：打开模板库

- `Ctrl + D`：下载ZIP
- `Ctrl + J`：下载JSON

## 常见问题

### Q1：ZIP文件无法解压？
A：确保下载完整，使用最新解压软件

### Q2：生成的项目无法运行？
A：确保已安装Node.js，运行 `npm install` 安装依赖

### Q3：历史记录不见了？
A：检查浏览器是否清除了LocalStorage

### Q4：页面显示异常？
A：尝试清除浏览器缓存或使用无痕模式
