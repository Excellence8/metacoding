# MetaCoding 使用指南

## 🎯 快速开始

### 第一步：访问应用
1. 打开浏览器访问 `http://localhost:5173`
2. 看到 MetaCoding 首页

### 第二步：选择模板
1. 点击 "📚 查看模板"
2. 选择一个模板（如 React + TypeScript）
3. 点击 "✅ 选择此模板"

### 第三步：生成项目
1. 自动跳转到生成器页面
2. 输入项目名称
3. 点击 "🚀 一键生成项目"
4. 等待 2 秒生成完成

### 第四步：导出项目
生成完成后可以选择：
- **📥 JSON 配置**: 下载项目配置文件
- **📦 ZIP 项目**: 下载完整的项目 ZIP 包
- **📋 复制配置**: 复制配置到剪贴板
- **🚀 复制命令**: 复制启动命令

## 📋 功能详解

### 1. 首页
- 项目介绍和快速入口
- 前往生成器和模板库的按钮
- 响应式设计，适配各种设备

### 2. 生成器页面
- **项目名称**: 自定义项目名称
- **模板显示**: 显示当前选择的模板
- **一键生成**: 核心功能按钮
- **结果展示**: 生成后的文件列表和命令

### 3. 模板库页面
- 4 种预设模板
- 每个模板都有图标和描述
- 点击模板直接跳转到生成器

### 4. 项目管理页面
- **项目统计**: 显示生成的项目总数和模板分布
- **历史记录**: 按时间顺序显示所有生成的项目
- **操作功能**: 重新生成、删除项目
- **清空历史**: 一键清除所有历史记录

### 5. 设置页面
- **主题设置**: 4 种颜色主题
- **编辑器设置**: 自动格式化、通知设置
- **语言设置**: 中文、英文、日文
- **字体大小**: 可调节的字体大小

## 🎨 主题系统

MetaCoding 支持 4 种主题：

### 🌞 浅色主题（默认）
- 明亮舒适的界面
- 适合白天使用
- 减少眼睛疲劳

### 🌙 深色主题
- 暗色背景，亮色文字

### 🔵 蓝色主题
- 蓝色系配色
- 专业、冷静的视觉感受
- 适合长时间工作

### 🟣 紫色主题
- 紫色系配色
- 创意、优雅的视觉感受
- 适合设计相关工作

## ⌨️ 键盘快捷键

### 全局快捷键
- `Ctrl + S`: 保存设置
- `Ctrl + H`: 返回首页
- `Ctrl + T`: 打开模板库
- `Ctrl + P`: 打开项目管理
- `Ctrl + ,`: 打开设置
- `Esc`: 关闭弹窗/返回

### 生成器页面快捷键
- `Ctrl + Enter`: 一键生成项目
- `Ctrl + D`: 下载 JSON 配置
- `Ctrl + Z`: 导出 ZIP 项目

## 📁 项目导出

### ZIP 项目包包含
1. **完整项目结构**: 开箱即用的 React 项目
2. **配置文件**: package.json、tsconfig.json、vite.config.ts
3. **源代码**: App.tsx、main.tsx、样式文件
4. **文档**: README.md、.gitignore
5. **HTML 文件**: index.html

### JSON 配置包含
- 项目基本信息
- 选择的模板
- 生成的文件列表
- 启动命令
- 生成时间戳

## 🔄 项目管理技巧

### 重新生成项目
1. 在项目管理页面找到历史项目
2. 点击 "🔄 重新生成"
3. 项目数据会自动加载到生成器
4. 可以修改配置后重新生成

### 批量管理
1. 使用 "🗑️ 删除" 单个删除项目
2. 使用 "🧹 清空历史" 批量删除所有项目
3. 项目统计会实时更新

## 🛠️ 高级功能

### 自定义模板
1. 修改 `src/App.tsx` 中的 `templates` 数组
2. 添加新的模板对象
3. 重启应用即可看到新模板

### 扩展导出功能
1. 修改 `src/utils/exportProject.ts`
2. 添加新的导出格式
3. 在生成器页面添加对应的按钮

### 主题定制
1. 修改 `src/utils/theme.ts`
2. 添加新的主题配置
3. 在设置页面会自动显示新主题

## 🔍 故障排除

### 常见问题

**Q: ZIP 导出失败怎么办？**
A: 确保已安装 JSZip 依赖，运行 `npm install jszip`

**Q: 主题切换不生效？**
A: 清除浏览器缓存或使用无痕模式

**Q: 项目历史不见了？**
A: 检查浏览器是否清除了 LocalStorage

**Q: 页面加载缓慢？**
A: 检查网络连接，或尝试刷新页面

### 获取帮助
1. 查看浏览器控制台错误信息
2. 检查控制台输出的详细错误
3. 根据错误信息搜索解决方案
4. 提交 Issue 寻求帮助
